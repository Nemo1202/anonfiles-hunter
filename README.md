# Installation
```
pip install colorama
pip install requests
pip install random
git clone https://github.com/fuckwbored/anonfiles-hunter
cd anonfiles-hunter
```
# Usage
```
python3 anonhunter.py <file-where-valid-links-will-be-writed>

ex:
python3 anonhunter.py output.txt
```
